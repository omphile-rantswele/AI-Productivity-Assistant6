# AI Workplace Productivity Assistant

## Project Overview

The AI Workplace Productivity Assistant is a web-based application designed to improve workplace efficiency by automating repetitive business tasks using Artificial Intelligence. The platform helps professionals generate emails, summarize meeting notes, plan tasks, conduct research, and interact with an AI-powered workplace assistant through a modern and intuitive interface.

This solution demonstrates practical AI implementation, prompt engineering, responsible AI practices, and productivity enhancement in a real-world business environment.

---

## Features

### Smart Email Generator

* Generate professional emails based on context

### Productivity Dashboard

## Setup Instructions

### Prerequisites

Before running the application, ensure you have:

* Node.js (v18 or later)
* npm or yarn
* OpenAI API Key

### Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/ai-workplace-productivity-assistant.git
```

2. Navigate to the project folder

```bash
cd ai-workplace-productivity-assistant
```

3. Install dependencies

```bash
npm install
```

4. Create a `.env` file

```env
VITE_OPENAI_API_KEY=your_openai_api_key
```

5. Start the development server

```bash
npm run dev
```

6. Open the application

```text
http://localhost:5173
```
